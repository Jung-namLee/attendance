⚡ 앱 소개

출퇴근 기록 앱은 매일 출근/퇴근 버튼 한 번으로 근무시간을 자동 기록하고, 구글 시트에 저장하는 앱이에요.

과제별 근무시간 관리, 달력 조회, 8h 미달 알림 기능을 제공해요.

📖 사용 방법
1. 기본 출퇴근 기록▸

① 홈 화면에서 ☀️ 출근 버튼 탭 → 출근 기록

② 퇴근할 때 🌙 퇴근 버튼 탭 → 퇴근 기록

③ 실근무시간은 점심 1시간이 자동으로 차감돼요

④ 하루 1회씩만 기록해주세요

2. 달력 화면▸

① 📅 달력 탭에서 월별 출퇴근 기록 확인

② 날짜 탭하면 상세 기록 확인 가능

③ 🔴 빨간 날짜 = 주말/공휴일

④ 붉은 배경 날짜 = 8h 미달 근무일

⑤ 날짜 안 숫자 = 부족/초과 시간 표시

3. 과제 관리▸

① 📁 과제 탭에서 과제명/시작일/종료일 등록

② 등록 후 해당 기간 달력에 과제 자동 표시

③ 종료일 변경 시 목표 근무시간 자동 재계산

④ 과제 진행현황에서 부족 시간 실시간 확인

4. 구글 시트 연동 방법▸

① 구글 시트 새로 만들기 (빈 스프레드시트)

② 상단 확장 프로그램 → Apps Script 클릭

③ 기존 코드 전체 삭제

④ 공유받은 코드 붙여넣기 → 저장(Ctrl+S)

⑤ 배포 → 새 배포 → 웹앱 선택

⑥ 액세스 권한: 모든 사용자 → 배포

⑦ 경고창 뜨면 Advanced → Allow 클릭

⑧ URL 복사 → 홈 화면 하단 🔗 구글 시트 연결에 붙여넣기 → 저장

5. 홈화면에 앱 추가 방법▸

① 크롬 브라우저로 앱 링크 접속

② 오른쪽 위 ⋮ → 홈 화면에 추가 탭

③ 추가 버튼 탭 → 완료!

※ 삼성 인터넷 말고 크롬으로 해주세요

⚠️ 사용 시 주의사항
