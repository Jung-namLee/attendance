📁 등록된 과제
⚡ 앱 소개

출퇴근 기록 앱은 매일 출근/퇴근 버튼 한 번으로 근무시간을 자동 기록하고, 구글 시트에 저장하는 앱이에요.

과제별 근무시간 관리, 달력 조회, 8h 미달 알림 기능을 제공해요.

📖 사용 방법
1. 기본 출퇴근 기록▸

① 홈 화면에서 ☀️ 출근 버튼 탭 → 출근 기록

② 퇴근할 때 🌙 퇴근 버튼 탭 → 퇴근 기록

③ 실근무시간은 점심 1시간이 자동으로 차감돼요

④ 하루 1회씩만 기록해주세요

2. 달력 화면▸

① 📅 달력 탭에서 월별 출퇴근 기록 확인

② 날짜 탭하면 상세 기록 확인

③ ✏️ 기록 수정 / 추가 버튼으로 깜빡한 날 시간 직접 입력 가능

④ 수정 후 ✅ 저장 탭하면 구글 시트도 자동 반영

⑤ 🗑 삭제는 두 번 탭으로 확인 후 삭제

⑥ 🔴 빨간 날짜 = 주말/공휴일

⑦ 붉은 배경 날짜 = 8h 미달 근무일

⑧ 날짜 안 숫자 = 부족/초과 시간 표시

3. 과제 관리▸

① 📁 과제 탭에서 과제명/시작일/종료일 등록

② 등록 후 해당 기간 달력에 과제 자동 표시

③ 종료일 변경 시 목표 근무시간 자동 재계산

④ 과제 진행현황에서 부족 시간 실시간 확인

4. 구글 시트 연동 방법▸

📌 PC에서 진행해주세요!


STEP 1. 구글 계정으로 로그인

👉 sheets.google.com 접속

👉 + 새 스프레드시트 만들기 (빈 시트)


STEP 2. Apps Script 열기

👉 상단 메뉴 확장 프로그램 → Apps Script 클릭

👉 새 탭으로 코드 편집 화면이 열려요


STEP 3. 코드 붙여넣기

👉 화면에 있는 코드 전체 선택(Ctrl+A) → 삭제

👉 아래 코드 복사 버튼 탭 → 붙여넣기(Ctrl+V)

👉 저장(Ctrl+S)


STEP 4. 웹앱으로 배포

👉 오른쪽 위 배포 → 새 배포 클릭

👉 ⚙️ 아이콘 클릭 → 웹앱 선택

👉 다음 사용자로 실행: 나

👉 액세스 권한: 모든 사용자

👉 배포 클릭


STEP 5. 권한 허용

👉 경고창이 뜨면 Advanced 클릭

👉 Go to ... (unsafe) 클릭

👉 Allow 클릭


STEP 6. URL 연결

👉 배포 완료 후 나타나는 URL 복사

👉 이 앱 홈 화면 하단 🔗 구글 시트 연결 칸에 붙여넣기

👉 저장 버튼 탭

👉 출근 버튼 눌러서 시트에 기록되면 완료! ✅

5. 홈화면에 앱 추가 방법▸

① 크롬 브라우저로 앱 링크 접속

② 오른쪽 위 ⋮ → 홈 화면에 추가 탭

③ 추가 버튼 탭 → 완료!

※ 삼성 인터넷 말고 크롬으로 해주세요
